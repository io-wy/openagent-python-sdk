# Demo task

`workspace/stats.py` ships with two known bugs that make
`workspace/test_stats.py` fail. Your job:

1. Read both files.
2. Run the test suite to see the failures.
3. Fix the bugs in `stats.py` *via `edit_file`* (do not rewrite the whole file).
4. Re-run the tests until everything passes.
5. Report which lines you changed and which tests now pass.

Constraints:

- Do not edit `test_stats.py` — the tests describe the contract.
- Do not introduce new dependencies.
- Make the minimal change for each bug.
